# templates/

## Purpose
Handlebars templates for auto-generating project files.

## Integration
Used by `clawforge init` to generate per-project CLAUDE.md.

## Key Files
- `CLAUDE.md.hbs` — Handlebars template listing all 35 tools with usage instructions

## Dependencies
- **npm**: `handlebars`

## Common Tasks
- Update template: edit `CLAUDE.md.hbs`, add new tool descriptions
