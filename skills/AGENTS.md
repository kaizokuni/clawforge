# skills/

## Purpose
Built-in skill definitions. Each subdirectory contains a SKILL.md with Antigravity-compatible frontmatter.

## Integration
Loaded by `src/tools/skills/loader.ts`. MCP tool: `skill_run`, `skill_list`
